## ⚙️ Como usar

* Necessario [NodeJS](https://nodejs.org/en/download/) instalado.
* Depois de clonar/baixar o repositório, use o *comando* `npm install` para instalar todas as dependências.
* Após isso, digite `npm start` para rodar a aplicação tem tempo real.
  * Para **transpilar** o código, use o comando `npm run build` ou apenas `tsc` no seu terminal.
